Read me for the VQE experiments in the "Exploiting symmetry in variational quantum machine learning" project.

- "ansatz_lib.py" : Here we constructed all the PQC ansatz used in the experiments.
- "BP_lib.py" : Here we implement the functions we use to evaluate the cost function.
- "hamiltonian_lib.py" : Here we built the Hamiltonians. 
- "main_BP_PQC_TFIM.py" : Code for the Barren Plateaus simulations of TFIM.
- "main_BP_PQC_XXX.py" : Code for the Barren Plateaus simulations of Heisenberg model.
- "main_VQE_scipy_TTT_LTFIM_plus.py" : Code for Tic-Tac-Toe LTFIM VQE simulation.
- "main_VQE_scipy_XXX.py" : Code for Heisenberg model VQE simulation.
- "main_VQE_scipy.py" : Code for TFIM VQE simulation.
- "myfunctions.py" : Here we include functions we use in the VQE simulations.
- "runscipy_TFIM.sh" : Script to run simulations for TFIM VQE.
- "runscipyTTTLTFIM_plus.sh" : Script to run simulations for Tic-Tac-Toe LTFIM VQE.
- "runscipyXXX.sh" : Script to run simulations for Heisenberg model VQE.
- "runTFIM_BP_AllP_jobarray.sh" : Script to run Barren Plateaus TFIM simulations.
- "runXXX_BP_AllP_jobarray.sh" : Script to run Barren Plateaus Heisenberg simulations.
